“云集客信息管理系统”服务使用协议

“云集客信息管理系统”（以下简称“平台”）是北京交通发展研究中心（以下简称“我方”）为工作体系内的单位、部门及相关工作人员提供的，用于完成云集客信息管理系统。

本平台用户由北京交通发展研究中心指定。指定用户在接受本协议并完成客户端安装后即可使用本平台。

第一款 用户申明

1、用户保证不利用本平台从事任何违反法律法规的活动。

2、用户保证不进行针对本平台的任何恶意行为。我方对于任何人任何破坏本平台的行为将保留追究其相应法律责任的权利。

3、用户承诺将仔细阅读指导用户操作的各种须知，同意将须知中的内容作为本使用协议的附属条款加以遵守，并按照其要求进行操作。

第二款 用户资料的提供及保密

1、 为保证本平台安全，用户本人须妥善保管登录平台的账号、USBKEY等。由于用户泄露账号或将USBKEY交与他人而导致信息泄露、破坏等不良后果，我方将保留追究其相应法律责任的权利。

2、 我方对用户个人信息以及用户上传保存的相关数据负有保密义务，不得向任何他人提供，但出现下列情形时除外：

(1) 用户书面请求并经我方同意公开其个人信息。

(2) 司法机关在符合国家相应法律和法规的情况下，调查索取用户个人信息。

3、 用户有权查阅并修改其个人信息。如果用户提供的资料不真实、不准确，我方有权终止服务，由此产生的损失由用户个人承担。

第三款 服务的方式

用户安装我方提供的客户端应用软件，并通过客户端软件使用我方服务系统提供的各类功能与信息。

第五款 协议的终止

1、交研中心收回或更换用户帐户；

2、用户单方要求终止服务；

3、 出现下列情况，我方有权随时终止本协议：

(1) 用户提供了不真实的帐号资料与信息；

(2) 用户利用本平台实施违反本协定、违法、犯罪活动。
